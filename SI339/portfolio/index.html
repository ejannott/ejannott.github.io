About My Projects

Below are some of my recent projects that I've been working on. These are from both my work in U of M's School of Information and professional work.

AutoMate

How can we build a stronger trust between autonomous cars and potentially dubious users through design? My team and I designed an autonomous vehicle interface to try and answer this question.

PyGame - Defend the City!

A small game that I coded in Python using Pygame. The goal was to learn how to use the modules within Pygame and code a compelling game using Python that was actually fun to play.

RMHC: Ann Arbor Mobile App

Helped the Ronald McDonald Family House in Ann Arbor design their app and bring it up to the new standards that RMHC set over the summer of 2017.
